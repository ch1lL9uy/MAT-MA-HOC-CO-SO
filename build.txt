- biên dịch từng file
g++ -std=c++17 -I include src/aes.cpp src/hash.cpp src/compressor.cpp src/hmac.cpp src/decryptionFormat.cpp -o MatMaHocCoSo.exe

- biên dịch toàn bộ trong src
g++ -std=c++17 -I include src/*.cpp -o MatMaHocCoSo.exe

- Sử dụng MakeFile:
    - Build tất cả: make
    - Build riêng lẻ: make Encrypt.exe hoặc make Decrypt.exe
    - Dọn file build: make clean
